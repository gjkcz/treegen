### Ovladani: ###

* w, s, a, d - dopredu, dozadu, doleva, doprava
* q, e - nahoru, dolu
* ↑, ↓ - pohled nahoru, dolu
* 1, 2 - přidat auto otáčení
* 3, 4 - otočit
